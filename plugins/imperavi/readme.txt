0.0.1	- 06.06.12 – релиз

<b>=== Описание плагина ===</b>
Плагин заменяет стандартный редактор LiveStreet на редактор Redactor (http://www.redactorjs.com). Распространяется "as is".
Плагин имеет возможность настройки следующих параметров:
- пути для загружаемых файлов;
- язык редактора;
- разрешенные типы файлов;
- максимально допустимые размеры разрешенных типов файлов;
- виды топиков, в которых производить замену стандартного редактора.

<b>===  Установка плагина ===</b>
Стандартная. Скопируйте каталог imperavi в каталог plugins и активируйте плагин по следующему пути http://ваш_сайт/admin/plugins

<b>=== Документация ===</b>
Плагин состоит из:
- дистрибутива редактора (/imperavi/templates/skin/default/js);
- файлов-скриптов, выполняющего замену стандартного редактора (/imperavi/templates/skin/default/js/replace_topic.js, replace_talk.js);
- файлов, обеспечивающих взаимодействие с LS.

<b>=== Настройка плагина и редактора ===</b>
Настройка плагина в файле /config/config.php
Настройка редактора согласно документации (http://redactorjs.com/ru/docs/) производится в файлах replace_topic.js, replace_talk.js

<b>=== Файлы, обеспечивающие взаимодействие с LS ===</b>
/imperavi/config/config.php - конфигурационный файл для настройки плагина. 
/imperavi/classes/actions/ActionImperavi.class.php - обеспечивает выполнение загрузки файлов в соответствии с файлом конфигурации config.php.
/classes/hooks/HookImperavi.class.php - обеспечивает подключение js-библиотек и css файлов редактора.
/imperavi/templates/skin/default/actions/ActionImperavi/upload_files.tpl, uploads_images.tpl
Шаблоны, обеспечивающие отображение сообщений об ошибках либо ссылки на файл после завершения загрузки.

<b>=== Доп. информация ===</b>
Настройки редактора производятся в файлах redactor_topic.js - для топиков, redactor_talk.js - для личных сообщений.

<b>=== Проблемы и их решения ===</b>
- Границы ячеек в таблице не отображаются.
Это настройки стиля шаблона. Изменяйте его, чтобы получить результат.
- Редактор не работает в браузере {ИмяБраузера}.
Обновите браузер до самой последней версии.
